## brandsome.dev + brandsome.app Strategy

Both are multi-tenant SaaS. Any startup can subscribe. Self-serve signup.

**First users:** Own projects (Trigyaa, KhabarOnline, JciAlumni, TrueValueEstate) → validate → open to public.

**brandsome.dev:**
- Logo + brand identity generator (fonts, colors, brand kit)
- CLI tool + AI Skills/MCPs (connectable to any AI coding tool like Codex, Claude Code)
- Competitors: logo.com, looka, brandmark

**brandsome.app:**
- Social media content creation (branded images, videos, Instagram reels)
- Uses brand kit from brandsome.dev
- Integrates with Postiz for scheduling
- Competitors: later phases

**Stack:** $1/mo — Vercel + Supabase + Cloudflare + Stripe + Clerk + Resend + PostHog
