# rakesh.tembhurne.com Playbook

Personal brand umbrella for 20+ micro-startups. Build-in-public. Indian solo-AI-founder voice.

## Project Status

**Status:** 🔴 Priority 1
**Live Site:** https://rakesh.tembhurne.com
**Repo:** https://github.com/rakeshtembhurne/rakesh-tembhurne-blog-next

## Concept

Personal site is the funnel for multiple ventures. Visitors become customers of Trigyaa, TradeMyBots, or clients of Dabble Lab. Build-in-public with transparent P&L.

## Tech Stack

- **Framework:** Next.js (App Router) with TypeScript
- **Styling:** Tailwind CSS v3
- **Content:** Contentlayer (markdown/MDX management)
- **Deployment:** Vercel
- **Analytics:** Umami + Google Analytics

## ICP & Positioning

Solo founder testing whether AI agents let one person run twenty projects. Indian market, rupee economics, agent-first workflow.

## Channel Strategy

1. **Twitter** — Primary. Build-in-public audience. 2-3 tweets/day.
2. **LinkedIn** — B2B funnel for Dabble Lab consulting. Long-form, 3/week.
3. **YouTube** — Experimental. Build walkthroughs + agent demos. 1/week.
4. **Blog** — SEO + depth. Weekly long-form build log.
5. **Newsletter** — Highest-intent channel. Weekly digest Sunday.

## Key Messages

- **1-line:** "Building 20+ micro-startups solo with AI agents. Public P&L."
- **3-line:** "I'm Rakesh. Solo founder building a portfolio of AI-first micro-startups. Every week I ship, break, document. No guru fluff — real numbers, real failures, real playbooks."

## Content Pillars

1. **Build Log** — Weekly what-I-shipped with numbers
2. **Skill Drops** — Open-source reusable Claude skills
3. **Revenue Updates** — Transparent monthly P&L
4. **Playbooks** — Repeatable systems with detail

## Voice & Tone

- **Formality:** 3/10
- **Humor:** 5/10 (self-deprecating, dry)
- **Emoji:** Sparingly. ⚡ for ship, 🔧 for debug, 📊 for numbers. Never 🚀🔥💯.

**Words we use:** shipped, broke, learned, ₹, actual, real, failed, pivoted
**Words we avoid:** crushing it, game-changer, 10x, revolutionary, INSANE, synergy
