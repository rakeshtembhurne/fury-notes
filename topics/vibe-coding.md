# Vibe Coding — Ship Fast, Iterate Faster

## Definition

**Vibe coding** = Build with AI assistance, fast iteration, don't overthink, ship and see what happens.

> "vibe code a revenue leaderboard in 4 hours, accidentally turn it into a marketplace, 1+ startups get acquired every day — you can just do things"

## Key Principles

1. **Speed over perfection** — Ship v0.001, not v1.0
2. **AI does the work** — Use Codex, Claude, etc. to code
3. **Accidents = opportunities** — Accidental features can become products
4. **Acquisition is a feature** — Build to sell, not just to build
