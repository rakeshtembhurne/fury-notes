## Trigyaa Visual Identity

Before generating images, know what Trigyaa looks like:

```
COLOR PALETTE:
- Primary: Ivory/Off-white (#FFFFF0 or #FAF9F6)
- Secondary: Natural linen beige (#C4B49A or similar warm neutral)
- Accent: Deep indigo (#2F3B6C — Indian indigo, not bright)
- Text: Charcoal (#333333)
- Skin tones: South Asian Indian, real models, not retouched

VISUAL STYLE:
- Minimal, clean, warm
- Natural light (golden hour or soft window)
- Real settings (home, office, commute, café — not studio)
- No heavy filters
- Indian women 25-40, natural poses
- Fabric texture always visible

WHAT WE AVOID:
- Studio gloss / high-key lighting
- White seamless backgrounds
- Heavy retouching / smoothing
- Western models
- Artificial looking AI skin (plasticky)
- Generic stock photo aesthetic
```

## Flux AI Image Prompt Structure

### Base Template for Trigyaa

```
[SUBJECT] in [SETTING], wearing [TRIGYAA PIECE DESCRIPTION],
[SHOOT STYLE — ambient, natural light, documentary],
[BACKGROUND DETAIL],
Indian woman, [APPROXIMATE AGE 25-35], [POSE/COMPOSITION],
natural skin texture, no heavy retouching,
[COLOR PALETTE — ivory, linen, indigo accents],
[intended social platform], [aspect ratio]

Negative prompt: white background, studio lighting, retouched skin,
plastic skin, oversaturated, text, watermark, logo, blurry
```

### Platform-Specific Aspect Ratios

```
Instagram Feed:       1:1 (square) or 4:5 (portrait)
Instagram Reel:       9:16 (portrait, 1080x1920)
Instagram Story:      9:16 (1080x1920)
Pinterest:           2:3 (portrait, 1000x1500)
Facebook:            1.91:1 (landscape, 1200x628)
WhatsApp:            1:1 (square)
Blog/SEO:            16:9 (landscape, 1200x675)
```

## Technical Parameters for Flux AI

### Recommended Settings

```
Model: FLUX.1 [dev] or FLUX.1 [schnell] (for speed)
Steps: 20-30 (quality vs speed tradeoff)
CFG Scale: 3.5-7 (lower = more natural, higher = more prompt-following)
Aspect Ratio: Match platform (see above)
Resolution: 1024x1024 base, upscale for print if needed
Seed: Random (for variations), lock seed for consistency across series

For skin quality:
- Use "natural skin texture" in prompt
- Use negative prompt "retouched skin, plastic skin, smoothing"
- Lower CFG helps maintain realistic skin
```

### Consistency Tips

```
For same model across multiple images:
- Lock seed for color/lighting consistency
- Use same setting/location description
- Use same subject description (age, skin, hair)
- Generate 3 variations, pick the best 1

For fabric color matching:
- Reference actual Trigyaa color codes from brand kit
- Use "color palette" description in prompt
- Match lighting across series shots
```

## Hermes Agent Integration

**Workflow for Hermes generating Trigyaa images:**

1. Hermes receives content plan (from [[trigyaa-content-calendar-30day]])
2. Hermes selects appropriate prompt template from this note
3. Hermes personalizes the prompt with:
   - Specific Trigyaa piece being featured
   - Current drop context (if applicable)
   - Season/timing (morning/evening light)
   - Platform the image is for
4. Hermes generates image via Flux AI
5. Hermes delivers to human review with caption draft
6. Human approves or requests revision

**Prompt template selection:**
```
New drop → Prompt 13 (teaser) or Prompt 14 (full reveal)
OOTD content → Prompt 1, 2, or 3
Behind the Seam → Prompt 4, 5, or 6
Customer UGC style → Prompt 7 or 8
Product flat lay → Prompt 9 or 10
Pinterest evergreen → Prompt 11 or 12
Care/how-to → Prompt 12
```
