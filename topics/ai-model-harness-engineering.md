# The AI Model Doesn't Matter Anymore

## Key Insights

1. **Benchmarks Are Misleading** - Frontier models achieve only 24% on real-world tasks despite scoring 90%+ on benchmarks. The metrics everyone argues about online don't reflect actual performance in production environments.

2. **Harness > Model** - The infrastructure wrapping the AI model matters more than the model itself. A harness manages: what the AI can see, what tools it can use, how it recovers from mistakes, and how it tracks long-running tasks.

3. **Model = Engine, Harness = Car** - The industry debates which engine (model) is best while ignoring that without a proper car (harness), the engine goes nowhere. Build the harness before debating models.

4. **Harness Engineering Is 2026's Key Skill** - Just as agents defined 2025, harness engineering will define 2026. Companies are already publishing guides on effective harnesses for long-running agents.

5. **Real Agent Differentiation Comes from Harness** - Projects like Codex and Cloud Code work not because of underlying model quality but because of the harness. This is where real engineering value and competitive advantage lives.
