# Productivity & Tools — Key Learnings

> Related: [[productivity-systems]] · [[n8n-ai-agents-workflows]] · [[ai-tools-2026]] · [[claude-code]]

## Tools Worth Knowing
- **LangExtract** — Open-source document extraction, free, reportedly better than $50K enterprise tools
- **Langtrace** — Open-source observability for LLM applications
- **ScreenPipe** — Captures screen activity for AI analysis and automation
- **OpenClaw** — AI-powered automation hub for workflows
- **Supabase** — Now has message queues (pgmq extension)

## Second Brain Architecture (Karpathy Method)
Three folders: raw, wiki, outputs + one CLAUDE.md schema file. Dump everything into raw without organizing. AI sorts it for you.

## AI Agent Workspace Structure
- `SOUL.md` — purpose and values
- `IDENTITY.md` — persona and communication style
- `USER.md` — user preferences and context
- `TOOLS.md` — available tools and capabilities
- `skills/` — modular skill definitions the agent can invoke

## "Second Brain as a Service" Business Model
- Charge $1,500-3,000 to build a client's knowledge base
- $300-500/month retainer for ongoing ingestion and health checks

## ScreenPipe + OpenClaw Workflow
1. Capture screen activity with ScreenPipe
2. Let AI analyze patterns
3. Generate automation ideas from observed workflows
4. Turn them into repeatable daily workflows inside OpenClaw

## Key Principles
- Templates and structured workflows save more time than any productivity hack.
- Automate what you do more than twice.
- Build habits around tools, not tools around habits.
- Real value = combining data + enrichment + templates into automated pipelines.
