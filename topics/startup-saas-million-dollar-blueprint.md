# Blueprint to Build a $1M SaaS From Scratch (Greg Isenberg + Rob Hoffman)

## Source
Greg Isenberg + Rob Hoffman (built $300K/mo and $20K-61K/mo SaaS businesses)

## Key Principles

### Start with "Boring" Niches
- Find unsexy, boring businesses that are making money
- Don't chase hype — chase cash flow
- Look for industries with no VC presence

### The Copy-Paste Playbook
1. Find a profitable SaaS in one niche
2. Copy the model to a new niche
3. Apply it to a different industry vertical

### Viral Loops (Reddit as #1 Acquisition Channel)
- Create a tool or landing page that does ONE thing extremely well
- Post it to relevant subreddits
- The tool does the selling — not ads
- Example: $0 → $30K MRR in 4 months via Reddit loop

### The "One Thing" Framework
- Your SaaS should do ONE specific thing incredibly well
- Don't try to be everything to everyone
- Solve one problem perfectly

### Landing Page Formula
- Hero: Specific outcome + time frame
- Subhero: How it works (3 bullets max)
- CTA: Start free trial
- Social proof: 3 testimonials

## Reverse-Engineered Winning SaaS Examples

### Notion式 AI Clone
- Take a popular tool
- Add AI capabilities
- Target a different niche
- Ship fast, iterate faster

### Browser-Based Tools
- No install required
- Easy sharing
- Low friction adoption

## Key Quote
> "Stop trying to build something original. Find what's already working and apply it to a new niche."
