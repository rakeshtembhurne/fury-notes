# OpenAI Codex

## Overview

OpenAI Codex is OpenAI's coding agent that powers GitHub Copilot and operates as a standalone coding agent. It provides free access via ChatGPT for code reviews and integrates with Claude Code.

---

## Codex for Claude Code

### Integration

OpenAI released an official Codex plugin for Claude Code:
- Pull Codex into Claude Code workflows
- Simple integration via plugin system
- Works with existing Claude Code setup

### Use Cases

1. **Code Reviews** - Second pair of eyes on code
2. **Complex Tasks** - GPT 5.4 for difficult problems
3. **Benchmarking** - Compare outputs across models
