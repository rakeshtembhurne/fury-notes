## Claude Code Agent Teams (Detailed)

### Key Points from Rocky's Research
- Sub-agents can't talk to each other; agent teams can
- Context is intentional, not automatic
- 3-5 teammates max (coordination overhead)
- Prompt approval pattern: teammates plan first, leader approves
- Shared memory file pattern for continuity

### Essential Plugins (6 only)
1. **Superpowers** — plan, write tests, sub-agents, self-review
2. **Frontend Design** — prevents generic AI look
3. **Code Review** — 5 parallel agents
4. **Security Guidance** — vulnerability scan
5. **Claude Memory** — persistent sessions
6. **Context7** — relevant docs

### Token Cost Strategy
- Use cheaper models (Sonnet) when possible
- Keep task lists focused
- Set explicit scope limits
