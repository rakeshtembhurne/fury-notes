# 10x Vibe Coding — Claude Code + Cursor + MCP Workflow

## Key Insights

- **Vibe coding is about multiplying yourself** — The goal is to build a portfolio of profitable apps using AI coding tools. The most effective vibe coders combine Claude Code (orchestration/planning), Cursor (fine-grained editing), and specialized MCPs for specific tasks.

- **MCP servers extend Claude Code's capabilities dramatically** — File system MCP, GitHub MCP, Slack MCP, Google calendar MCP, and others let Claude interact with your full tech stack. Riley Brown demoed 31 tools connected to his Claude instance in under an hour using NAN's native MCP trigger.

- **The key word that makes Claude Code work harder: "think harder"** — Triggering deeper reasoning in Claude Code produces significantly better outputs. Use it when outputs feel shallow or generic.

- **Cursor has a specific advantage for UI iteration** — Cursor's composer makes it easy to make targeted changes to specific files without re-running entire workflows. Use Claude Code for planning + backend, Cursor for targeted UI refinements.

- **SuperDesign (superdesign.dev) is the "Cursor for design"** — An interactive canvas where you can watch AI create/modify UI iterations in real time. Now works with your existing Claude Code subscription. Use it to iterate on designs visually before committing to code.

## Summary

The 10x vibe coder workflow: Claude Code for planning and orchestration, Cursor for targeted edits, MCPs for full tech stack access (Slack, GitHub, calendar, etc.). Use "think harder" to get deeper reasoning. SuperDesign bridges the gap between design and code. The combination lets a single builder ship full products without a team.
