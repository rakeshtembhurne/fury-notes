## The Start Frame Principle

> "Your start frame is responsible for 80% of your result. If you have a bad start frame, no amount of prompt engineering is going to save it."

**Mistakes to avoid:**
- Using blurry/low-quality start frames
- AI-looking plasticky skin in start images
- Garbage in = garbage out

**How to create great start frames:**
1. Use quality image generation (Soul 2, Nana Banana Pro)
2. Describe: who, where, what's happening, camera move, mood
3. Use character sheets + tagging for consistency

## Prompting Like a Movie Director

**The 5 elements of every prompt:**
1. **Who** — subject/person
2. **Where** — location/setting
3. **What's happening** — action/event
4. **Camera move** — dolly, zoom, orbital, etc.
5. **Mood** — emotional quality of the scene

**Key insight:** Stop thinking like a prompt engineer. Start thinking like a movie director.

> "The biggest mistake I see is typing a long prompt and hitting generate. That's text to video. And it's basically a slot machine. You have zero control over the composition."

## 38 Camera Moves for Cinematic AI Videos

### Categories:
1. **Dolly moves** — camera on wheeled platform moving through space (forward/back)
2. **Zoom techniques** — lens manipulation for dramatic effect
3. **Orbital movements** — camera rotating around subject
4. **Vertical movements** — crane shots for dramatic reveals
5. **Subject-based movements** — camera follows the action

### Pro Tips:
- Be specific: "slow" vs "fast" changes everything
- Combine techniques: dolly + zoom = vertigo effect
- Match mood: action = handheld, drama = slow moves
- Consider subject: people vs objects need different treatment
- Test variations: small prompt changes = big differences

## AI YouTube Workflow (Magic Light vs Manual)

### Magic Light (5-15 min)
- Fast turnaround
- Good quality
- High scalability
- Best for: beginners, scaling

### Manual Process (2-4 hours)
- Premium/cinematic quality
- Low scalability
- Best for: courses, high-end content

### Workflow Steps:
1. **Find validated ideas** — use YouTube in Incognito mode
2. **Extract transcripts** — from top videos (replace /watch with /transcript)
3. **ChatGPT analyzes patterns** — viral hooks, structure, psychological triggers
4. **Create video** — Magic Light or Manual based on needs
5. **Consistency > Quality at first**
