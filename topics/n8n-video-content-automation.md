# n8n Video & Content Automation

## Overview
n8n enables fully automated video content pipelines - from generation to publishing across multiple platforms.

## Video Content Automation Systems

### 1. Shorts/Reels Automation (100+ clips from 1 video)
- AI analyzes long-form video content
- Auto-clips with proper framing (speaker in place)
- Auto-generates subtitles
- Publishes to TikTok, Instagram, YouTube Shorts, LinkedIn
- Rate: ~$15/minute manual clipping → free with automation

### 2. AI Art Factory (TikTok Content)
- Auto-generates digital wallpapers using AI image models
- ChatGPT image model integration
- Auto-adds music
- Scheduled posting (daily/hourly)
- 122M+ view accounts achieved
- Monetization: Sell wallpapers, affiliate, brand deals

### 3. Longform YouTube Video Generator
- Hourly automated video creation
- Workflow: Idea generation → Prompt creation → Visuals → Voiceover → Music → Subtitles → Publish
- Niches: Motivational, stoicism, sleep stories, horror, children's stories, trivia
- Example: 1.35M subscribers with only 33 videos

### 4. Newsletter to Video Pipeline
- Forward newsletter to automation
- AI generates 3 unique short videos with:
  - Voiceover
  - Smart moving images
  - Music
  - Subtitles
  - Captivating titles
- Auto-posts to Instagram, TikTok, YouTube, Facebook, LinkedIn

### 5. Talking Baby Podcast
- Select topic + baby ethnicity
- One-click generation
- Auto-post to YouTube + social platforms
- Viral format on TikTok/Instagram/YouTube

## Content Repurposing Workflow

YouTube Video → Transcript → Multi-format output:
1. **Sales Enablement**: Battle cards, competitor analysis
2. **Social Media**: LinkedIn, Instagram, Twitter, Facebook posts
3. **Internal Comms**: Email summaries, Slack updates
4. **Knowledge**: AI summaries

## Key n8n Nodes for Video

- **HTTP Request**: Fetch video content, APIs
- **AI Agent**: Decision making, content planning
- **Code Node**: HTML stripping, data processing
- **RSS Feed**: Content sources
- **Vector Store**: Content indexing/search
