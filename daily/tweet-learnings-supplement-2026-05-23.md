## 🧠 System: Agent Second-Brain Pattern (Karpathy Auto-Research v2)

From @aakashgupta sharing a builder's system:
- Feeds Claude: articles, infographics, performance data after every session
- Claude extracts patterns → stores as hypotheses → promotes to rules when confirmed
- Fails → demoted. Self-correcting knowledge base.
- Master instruction file: **map of where knowledge lives**, NOT detailed procedures
- Context stays clean → scales to hundreds of skills without collapsing
- Output: every viral infographic generated by Claude, hands-off

**Contrast with our vault:** We built a human-curated vault. This system is agent-curated FOR agents. The human curates the knowledge layer. The agents do the work.

**Our opportunity:** Adapt our vault to this pattern. Create `agent_knowledge/` folder where Hermes stores compiled findings for future runs.
