## Session 2 — Evening Research (May 19, Evening)

**Duration**: ~30 minutes
**Agent**: Fury (autonomous research)
**Topic**: Postiz deep dive + AI video tools comparison for Brandsome.app

### Research Done

1. **Postiz research** (postiz.com)
   - Pricing tiers: $29-99/mo
   - Built-in AI image + video generation (3-60 videos/mo by plan)
   - Agentic integrations: ChatGPT, OpenClaw, Claude Code, n8n, Zapier, Public API
   - Self-hostable (AGPL-3.0, Docker)
   - Already integrates with Hermes/OpenClaw via MCP
   - Key insight: Postiz already has AI video gen — Brandsome.app shouldn't duplicate this

2. **Runway pricing** (runwayml.com/pricing)
   - Standard ($12/mo): 625 credits, Kling 3.0 Pro access, all 3rd party models
   - Pro ($28/mo): 2,250 credits, 500GB storage, custom voices
   - Key insight: Runway Standard = cheapest way to get Kling 3.0 Pro

3. **Pika pricing** (pika.art/pricing)
   - Basic ($8/mo): 80 credits, all resolutions
   - Standard ($28/mo): 700 credits
   - Key insight: Good for short social clips (5-10s), credit-based pricing

4. **Kling AI** (kling.ai)
   - Upgraded to 3.0 series (May 2026)
   - API docs not publicly accessible (redirects)
   - Access via Runway Standard (included) or direct API

### Key Findings

1. **Postiz = scheduling layer** — Don't build this. Integrate it.
2. **Runway Standard ($12/mo) = best video quality** — Includes Kling 3.0 Pro
3. **Brandsome.app focus: brand kit + template system** — Not AI video gen
4. **Recommended stack: Postiz ($39/mo) + Runway Standard ($12/mo) = $51/mo**
5. **Self-host Postiz** on existing infrastructure = free forever (no cloud fees)
